# How to work on the reckoning

Epistemological guidance for working on the documents in this folder. 

---

## [MENEHUNE] discipline

When the project uses geometric or computational language — forcing function, projection distance, coordinate change, tiling — each usage must be either a proved statement or an explicit conjecture marked [MENEHUNE]. Do not let analogies remain analogies. 

If you are unsure whether a claim is established, mark it. If you are unsure whether a step is complete, mark it. The test: if someone removed the analogy and asked "what is the theorem?", could you state one? If yes, do so.
